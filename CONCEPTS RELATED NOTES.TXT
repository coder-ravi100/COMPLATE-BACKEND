#2.DJANGO FRAMEWOORKS
    ⛔What is Django
        🔺High-level Python Web Framework
        🔺Designed for fast. secure. scalable. development
        🔺Open-source & community-driven
        🔺follows MVT (Model View Template) architecture
        🔺Motto : The web Framework for perfectionists with deadlines
    
    ⛔Key Features
        🔺Batteries include-built-in tool for authentication. admin panel. ORM. etc..
        🔺Scalable-From Small Apps to large-scale platforms
        🔺Secure - Protects against SQL injection , XSS, CSRF
        🔺Cross-platforms-works  on windows, mac, Linux
        🔺Strong community Support - Large Ecosystem of plugins and tutorials
    
    ⛔Why Use Django?
        🔺Fast  development- build projects quickly
        🔺Security-inbuilt Security best practices
        🔺Scalability - Handles high trafic and large database
        🔺Build-in Admin Panel-Manage data without extra coding
    
    ⛔Real World Use
        🔺Instagram - Social media
        🔺Pinterest - image Sharing platforms
        🔺Mozilla - Add-one Store


    ⛔MVT architecture In Django
        🔺What is MVT?
        🔺Components of MVT
        🔺MVT workflow
     
     ⛔MVT workflow Step by Step
        🔺User sends a request (URL)
        🔺URL Dispatcher maps request to View
        🔺view interacts with Model to get/save data
        🔺view sends data to Template
        🔺Template renders HTML
        🔺HTML is send as response to user
